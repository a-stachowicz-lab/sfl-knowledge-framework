# SFL-Wissensrahmen

## Purpose
Dieses Repository dient als kollaborativer, rollenbasierter Wissensrahmen für
Strukturell-Funktionale Schichten (Structural Functional Layers, SFL).

Der Fokus liegt auf:
- materialagnostischer Wissensorganisation
- funktionalen Rollen statt Materialien
- expliziten Konsistenz- und Entkopplungsregeln

## Scope
- Wissensstruktur, nicht Produktdesign
- Architektur- und Regelwissen, keine Rezepte
- Vorstufe zu Simulation, Optimierung und KI-Tools

## Relation to Publications
Dieses Repository ergänzt folgende Preprints:
- *Material-Agnostic Functional Architecture for Multilayer Composite Systems*
- *Constraint-Guided Generative System Design Based on SFL*
- *Ein kollaboratives, rollenbasiertes Wissensmodell für SFL*

## Governance
Beiträge müssen:
- einer klaren SFL-Rolle zugeordnet sein
- materialagnostisch formuliert werden
- bestehende Konsistenzregeln respektieren
